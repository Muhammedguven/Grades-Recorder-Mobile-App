# Grades Recorder Mobile App
It is a simple mobile app which is coded in Xamarin

			Usage

You should add grades and percentile of your exams. 
Then, the app calculate the average and list all of your lessons.
If you want to remove a lesson details, hold down to the lesson then click “Sil” button.

		       Warning
There is no SQL connection. Data are kept in objects. 
So, when you turn off the app, all of the data are going to disappear.
